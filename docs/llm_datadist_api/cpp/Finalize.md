# Finalize

## 产品支持情况

| 产品 | 是否支持 |
| --- | --- |
| Atlas A3 训练系列产品/Atlas A3 推理系列产品 | √ |
| Atlas A2 推理系列产品 | √ |
| Atlas A2 训练系列产品 | x |

## 函数功能

LLM-DataDist的资源释放函数。

## 函数原型

```
void Finalize()
```

## 参数说明

无

## 返回值

无

## 异常处理

无

## 约束说明

- 需要和[Initialize](Initialize.md)配对使用。
- 分离部署下，当增量集群去全量集群拉取kv的时候，需要保证全量集群在同步完kv的数据之后才调用该接口；否则会出现失败。
- 分离部署下，当全量集群在同步kv数据给增量集群的过程中，需要保证增量集群在全量集群同步完kv数据之后调用该接口，否则会出现失败。
- 该接口不能和其他接口并发调用。
