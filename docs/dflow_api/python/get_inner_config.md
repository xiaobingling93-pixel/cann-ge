# get\_inner\_config

## 产品支持情况

| 产品 | 是否支持 |
| --- | --- |
| Atlas A3 训练系列产品/Atlas A3 推理系列产品 | √ |
| Atlas A2 训练系列产品/Atlas A2 推理系列产品 | √ |

## 函数功能

获取内部配置对象，被[set\_output](set_output.md)或者[set\_multi\_outputs](set_multi_outputs.md)调用。

## 函数原型

```
get_inner_config(self)
```

## 参数说明

无

## 返回值

内部配置对象。

## 异常处理

无

## 约束说明

无
