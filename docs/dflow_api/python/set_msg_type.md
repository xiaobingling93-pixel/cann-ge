# set\_msg\_type

## 产品支持情况

| 产品 | 是否支持 |
| --- | --- |
| Atlas A3 训练系列产品/Atlas A3 推理系列产品 | √ |
| Atlas A2 训练系列产品/Atlas A2 推理系列产品 | √ |

## 函数功能

设置FlowMsg的消息类型**。**

## 函数原型

```
set_msg_type(self, msg_type) -> None
```

## 参数说明

| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| msg_type | 输入 | FlowMsg的消息类型。用户自定义类型范围为：[1204, 65535) |

## 返回值

无

## 异常处理

无

## 约束说明

无
