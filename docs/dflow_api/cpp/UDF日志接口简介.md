# UDF日志接口简介

flow\_func\_log.h提供了日志接口，方便flowfunc开发中进行日志记录，日志相关场景如下。

**表 1**  日志分类

| 日志类型 | 使用场景 | 日志级别 | 对应的日志宏 |
| --- | --- | --- | --- |
| 运行日志 | 系统运行过程中的异常状态、异常动作、系统进程运行过程中的关键事件和系统资源占用的相关信息等需要记录运行日志。 | ERROR | [运行日志Error级别日志宏](运行日志Error级别日志宏.md) |
| INFO | [运行日志Info级别日志宏](运行日志Info级别日志宏.md) |  |  |
| 调试日志 | 以下场景（包括但不限于这些场景）需要记录为调试级别的日志：<br>- 接口调用、函数调用等所有调用的入口、出口<br>- 操作入口处和设置预置条件<br>- 定时器启动、超时<br>- 状态设置、状态迁移条件判断前后<br>- 业务相关资源统计、业务处理出入口、性能计算统计<br>- 所有处理失败、异常等<br>调试级别日志记录的是代码级的信息，用于开发人员定位问题。 | ERROR | [调试日志Error级别日志宏](调试日志Error级别日志宏.md) |
| WARN | [调试日志Warn级别日志宏](调试日志Warn级别日志宏.md) |  |  |
| INFO | [调试日志Info级别日志宏](调试日志Info级别日志宏.md) |  |  |
| DEBUG | [调试日志Debug级别日志宏](调试日志Debug级别日志宏.md) |  |  |
