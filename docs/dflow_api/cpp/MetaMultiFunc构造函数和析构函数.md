# MetaMultiFunc构造函数和析构函数

## 产品支持情况

| 产品 | 是否支持 |
| --- | --- |
| Atlas A3 训练系列产品/Atlas A3 推理系列产品 | √ |
| Atlas A2 训练系列产品/Atlas A2 推理系列产品 | √ |

## 函数功能

用户继承该类进行自定义的多func处理函数的编写。在析构函数中，执行释放相关资源操作。

## 函数原型

```
MetaMultiFunc()
~MetaMultiFunc()
```

## 参数说明

无。

## 返回值

返回MetaMultiFunc类型的对象。

## 异常处理

无。

## 约束说明

无。
