# Warn

## 产品支持情况

| 产品 | 是否支持 |
| --- | --- |
| Atlas A3 训练系列产品/Atlas A3 推理系列产品 | √ |
| Atlas A2 训练系列产品/Atlas A2 推理系列产品 | √ |

## 函数功能

记录Warn级别日志。

## 函数原型

```
virtual void Warn(const char *fmt, ...) __attribute__((format(printf, 2, 3))) = 0
```

## 参数说明

| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| fmt | 输入 | 日志格式化控制字符串。<br>取值范围：非空。 |
| ... | 输入 | 可选参数<br>与fmt匹配的参数。 |

## 返回值

无。

## 异常处理

日志流控后日志不会记录。

## 约束说明

无。
