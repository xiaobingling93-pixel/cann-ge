# GetTensorList

## 产品支持情况

| 产品 | 是否支持 |
| --- | --- |
| Atlas A3 训练系列产品/Atlas A3 推理系列产品 | √ |

```
virtual const std::vector<Tensor *> GetTensorList() const
```

## 参数说明

无

## 返回值

所有的Tensor指针列表。

## 异常处理

无。
