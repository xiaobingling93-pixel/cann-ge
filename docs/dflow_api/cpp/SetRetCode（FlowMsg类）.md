# SetRetCode（FlowMsg类）

## 产品支持情况

| 产品 | 是否支持 |
| --- | --- |
| Atlas A3 训练系列产品/Atlas A3 推理系列产品 | √ |
| Atlas A2 训练系列产品/Atlas A2 推理系列产品 | √ |

## 函数功能

设置FlowMsg消息中的错误码。

## 函数原型

```
void SetRetCode(int32_t retCode)
```

## 参数说明

| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| retCode | 输入 | 错误返回码。 |

## 返回值

无

## 异常处理

无。

## 约束说明

无。
