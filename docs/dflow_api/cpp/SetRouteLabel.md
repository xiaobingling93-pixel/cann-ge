# SetRouteLabel

## 产品支持情况

| 产品 | 是否支持 |
| --- | --- |
| Atlas A3 训练系列产品/Atlas A3 推理系列产品 | √ |
| Atlas A2 训练系列产品/Atlas A2 推理系列产品 | √ |

## 函数功能

设置route的标签。

## 函数原型

```
virtual void SetRouteLabel(uint32_t routeLabel) = 0
```

## 参数说明

| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| routeLabel | 输入 | route的标签，取值为0表示不使用。 |

## 返回值

无。

## 异常处理

无。

无。
